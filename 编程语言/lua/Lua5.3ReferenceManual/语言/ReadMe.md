# 关键字
```lua
    and       break     do        else      elseif    end
     false     for       function  goto      if        in
     local     nil       not       or    repeat    return
    then      true      until     while
```

# 控制语句
循环语句
```lua
while exp do block end
repeat block until exp
for exp do end
```
条件语句
```lua
if exp then
[elseif exp then]
[else]
end
```
跳出
```lua 
goto return
```
